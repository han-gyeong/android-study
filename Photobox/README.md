## 전자액자 만들기

유저의 핸드폰에서 사진을 받아와서 '전자액자 실행하기' 를 누르면 1초 간격으로 그림을 슬라이드 쇼 방식으로 표시

- 안드로이드로 권한 받아와서 권한 사용하기
- 뷰 애니메이션 사용해보기
- 액티비티 라이프사이클 사용해보기
- 컨텐츠 프로바이더에서 SAF(Storage Access Framework) 사용해보기
